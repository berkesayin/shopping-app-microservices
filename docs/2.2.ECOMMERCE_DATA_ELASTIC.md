# Elastic's E-Commerce Orders Data 

Elastic provides sample datasets to use in projects. To get that, open [Kibana Server]("https://localhost:5601") and go to: `Menu` &rarr; `Management` &rarr; `Integrations`. Search for `Sample Data` there. Open and add `Sample eCommerce Orders` data.

`Kibana Sample Data eCommerce` is added as `Data View` at `Discover`.

![img](./img/1.sample-ecommerce-data.png)

The dashboard visualizes the orders.

![img](./img/2.sample-ecommerce-data.png)

Use `Visualize Library` to get meaningful results about orders.

![img](./img/3.sample-ecommerce-data.png)
